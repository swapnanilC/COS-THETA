# Security Policy

## Supported Versions
This project is a frontend-only website and is not actively maintained for external use.
No versions are currently supported for security updates.

## Reporting a Vulnerability
If you discover a security issue related to this repository, please report it responsibly.

- Do NOT open public issues.
- Do NOT submit pull requests.
- Contact the repository owner directly if necessary.

## Scope
This repository contains only frontend code and does not handle:
- User authentication
- Sensitive data
- Payments or transactions
- Backend services

## Contributions
External contributions, modifications, or reuse of this code are not permitted.

## Disclaimer
This project is provided as-is for showcase purposes only.
The owner is not responsible for misuse or security issues arising from third-party deployments.
